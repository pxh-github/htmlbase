# 引用：
#### 图片引用
```
<img src="(地址)">
```
# 标签：
#### 标题标签
```
<h1>123</h1>
<h2>123</h2>
```
…………(越小字越大)
#### 段落标签
```
<h></h>
```
#### title标签
```
<title><\title>
```
#### 换行标签
```
<br/>
```
# 快捷键：
#### 快速复制一行
```
shift+alt+pgup/pgdn
```

#### 选中相同元素
```
ctrl+d
```
#### 添加多个光标
```
ctrl+alt+pgup/pgdn
```
#### 查找并替换某一元素
```
ctrl+h
```
#### 跳转到指定行
```
ctrl+g
```
#### 选中区块
```
shift+alt+drag the mouse
```
#### 调整页面大小
```
ctrl + '+'/'-'
```
